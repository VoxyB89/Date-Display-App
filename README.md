# Date Display App

Simple web tool to count days, months, years from today until any future date.  
Show results on screen or download them as text files.

Live: https://voxyb89.github.io/DateDisplayApp/

### Features
- Pick a date → see full calendar dates, just days, months, or years
- Clean output with copy buttons
- Download as .txt files (one click)
- Light/dark mode toggle
- Mobile friendly
- No data leaves your browser

### How to run locally
1. Clone or download this repo
2. Open `index.html` in Chrome/Firefox/Edge
3. Pick a date and play around

### Why I made this
Needed a quick way to list dates for planning stuff (events, countdowns, lists) without spreadsheets.  
Hope it's useful for someone else too.
