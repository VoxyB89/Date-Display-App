Quick Start Guide

  1. Pick any future date with the picker above.
  2. Hit one of the "Show" buttons — results appear in the row below.
  3. Scroll horizontally if needed (or vertically on phone).
  4. Use "Download" buttons to save as .txt files.
  5. Click "Copy" on any box to copy the text fast.
  6. Switch light/dark mode with the icon in the header.
